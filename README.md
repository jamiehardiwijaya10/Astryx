# 🌫️ Morivelle

> Sebuah game CLI berbasis story, misteri, dan city-building ringan dengan elemen puzzle dan combat.

## 📖 Tentang Game

**Morivelle** adalah game berbasis CLI yang menggabungkan:
- narasi misteri
- investigasi
- puzzle (riddle)
- dan combat sederhana

Game ini berfokus pada eksplorasi sebuah kota yang telah hancur secara misterius.
